# AgentKit 沙箱工具

## 概览

AgentKit 工具提供强大的企业级功能，可增强 AI 智能体安全高效地执行复杂任务的能力。该套件包括两个主要工具：

- **AgentKit 代码解释器 (Code Interpreter)**
- **AgentKit 浏览器工具 (Browser Tool)**

---

## AgentKit 代码解释器

AgentKit 代码解释器允许 AI 智能体在安全的托管沙箱环境中执行代码。这使得智能体能够执行超出标准语言模型能力的复杂计算、数据分析和处理任务。

### 主要特性

- **安全的代码执行**: 代码在隔离的沙箱环境中执行，确保在访问内部数据源时的安全性。
- **完全托管的解决方案**: 与 LangChain、LangGraph 等流行的智能体框架无缝集成。
- **高级配置**: 支持大文件的输入和输出，允许进行复杂的数据操作。
- **多语言支持**: 内置了对多种编程语言（包括 Python 和 JavaScript）的运行时环境支持。

### 核心优势

- **提升智能体准确性**: 使智能体能够执行精确的计算和复杂的数据处理，从而获得更可靠的结果。
- **企业级安全**: 通过在隔离和受控的环境中运行代码，满足严格的安全要求。
- **高效的数据处理**: 能够通过引用文件和执行复杂脚本来处理大规模数据。

---

## AgentKit 浏览器工具

AgentKit 浏览器工具赋予 AI 智能体与网页交互的能力，可自动执行通常需要人工手动干预的任务。它能够浏览网站、填写表单并实时提取信息。

### 主要特性

- **模型无关的灵活性**: 设计上兼容来自各种主流 AI 模型的命令语法，提供了广泛的灵活性。
- **企业级安全**: 提供强大的安全措施，包括虚拟机级别的隔离和通过 VPC 连接的能力。
- **全面的审计能力**: 为所有浏览器命令和操作提供强大的日志记录功能，确保完全的可追溯性。

### 核心优势

- **端到端的自动化**: 使 AI 智能体能够自动执行复杂的基于 Web 的工作流，例如数据录入、信息收集和多步骤在线流程。
- **增强的安全性**: 通过广泛的安全功能和详细的审计追踪，满足严格的企业安全与合规要求。
- **实时交互**: 允许智能体与实时网页进行交互，以获取最新信息。

---

## 应用场景

- 在安全环境中进行复杂的数据分析和可视化。
- 自动化的 Web 交互，用于表单填写、数据提取和流程自动化。
- 大规模数据处理和监控。
- 在企业环境中为 AI 智能体安全地执行代码。
